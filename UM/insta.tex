\experience{Course Project: Instagram Clone}{2019-9-0}{2019-10-0}{
Developed a photo sharing website. Support posting, liking, commenting, and following.
\begin{itemize}
    \item Used React for frontend.
    Implemented React components for posts, likes, and comments.
    Wrote pages for home feed, user profiles, following, followers, exploring, and account actions.
    \item Used Flask to serve the React app and REST APIs that handle Ajax calls.
    Stored data using SQLite.
    Deployed the website on AWS using Gunicorn and Nginx.
\end{itemize}
